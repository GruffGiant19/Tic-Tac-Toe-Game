# Tic Tac Toe Game

A classic Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript. Play against a friend or enjoy a quick game on your web browser.

## Features

- **Two-Player Gameplay**: Play as X and O against another player
- **Win Detection**: Automatically detects winning conditions
- **Restart Button**: Easily reset the game to play again
- **Responsive Design**: Clean, modern interface with responsive styling
- **Real-time Status**: Displays whose turn it is and announces the winner

## How to Play

1. Open `index.html` in your web browser
2. Player X goes first
3. Click on any empty cell to place your mark
4. Players alternate turns between X and O
5. First player to get three in a row (horizontally, vertically, or diagonally) wins
6. Click the "Restart" button to start a new game

## Winning Conditions

A player wins by getting three of their marks in a row:

- Horizontal rows
- Vertical columns
- Diagonal lines

## Browser Compatibility

Works with all modern browsers that support:

- HTML5
- CSS3
- ES6 JavaScript

## Getting Started

Simply open the `index.html` file in any web browser to start playing!

## Technologies Used

- **HTML5**: Game structure and grid layout
- **CSS3**: Styling and visual design
- **JavaScript (Vanilla)**: Game logic, player turns, and win detection
- **Google Fonts**: Permanent Marker font for styling
